How Can I Add a Place?

In order to add a place to 'your places' two things need to happen. First you need to be logged in, and second the place you are trying to add needs at least one review. If you are reading this message that means there are no reviews yet for this place. You can be the first one to leave a review!

Why Should You Add a Place?

Adding a place to 'My Places' will send you updates if we notice any changes in normal reviews. This can be helpful because if one of your places has a drastic change in reviews you may want to visit another location.
